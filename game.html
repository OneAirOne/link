Code GitHub

Examen final de la piscine de C, développé en une journée.

Technologie utilisée : C

L'algorithme affiche un sapin à n segments, chaque segment comportant 3 + n branches. La largeur d'une branche est de 2 étoiles de plus que la branche précédente. Lors d'un changement de segment la largeur est celle de la branche supérieure
